Dissertation Focus

Development of a Real-Time Prompt Gamma Imaging System for Proton Beam Range Verification

This work involves designing and optimizing a detector system for **prompt gamma ray** emission to verify the Bragg Peak position *in vivo* during patient treatment, using detector physics and signal processing.

  • Key Methods: Detector Modeling, High-Purity Germanium (HPGe) Simulations, Signal Processing, GATE/GEANT4.
  • Tools: C++, ROOT, Python/NumPy, Clinical Treatment Planning System (TPS) integration.
Status: Detector Simulation & Prototyping Phase (Expected Defense: 2026)
